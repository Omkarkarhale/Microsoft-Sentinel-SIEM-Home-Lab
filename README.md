# Microsoft Sentinel SIEM Home Lab – Microsoft Entra ID Log Integration

## Overview

This project demonstrates how to integrate **Microsoft Entra ID** with **Microsoft Sentinel** to collect and analyze authentication logs. By enabling diagnostic settings, Microsoft Entra ID sends **Audit Logs** and **Sign-in Logs** to a Log Analytics Workspace, allowing security analysts to monitor user activities using Kusto Query Language (KQL).

---

## Prerequisites

* Microsoft Azure Subscription
* Microsoft Sentinel enabled
* Log Analytics Workspace connected to Microsoft Sentinel
* Microsoft Entra ID administrative permissions

---

## Implementation Steps

### Step 1: Install the Microsoft Entra ID Solution

1. Open **Microsoft Sentinel** in the Azure portal.
2. Navigate to **Content Hub**.
3. Search for **Microsoft Entra ID**.
4. Select the solution and click **Install**.
<img src=1.png>
---

### Step 2: Configure Microsoft Entra ID Diagnostic Settings

1. Open **Microsoft Entra ID** in the Azure portal.
2. Navigate to **Monitoring & Health** → **Diagnostic Settings**.
3. Click **Add Diagnostic Setting**.
4. Select the following log categories:

   * **AuditLogs**
   * **SigninLogs**
5. Under **Destination Details**, select **Send to Log Analytics Workspace**.
6. Choose the Log Analytics Workspace associated with your Microsoft Sentinel instance.
7. Click **Save** to enable log forwarding.
<img src=2.png>
---

### Step 3: Verify Log Ingestion

1. Return to **Microsoft Sentinel**.
2. Select **Logs** from the left navigation pane.
3. Execute the following KQL query to retrieve sign-in events for a specific user:

```kql
SigninLogs
| where Identity == "Omkar Karhale"
```

## Expected Output

The query displays all Microsoft Entra ID sign-in events generated by the specified user, including information such as:

* User Identity
* Sign-in Status
* Timestamp
* IP Address
* Location
* Authentication Method
* Client Application
* Device Information

This confirms that Microsoft Entra ID logs are successfully ingested into Microsoft Sentinel and are available for security monitoring, threat hunting, and incident investigation.
